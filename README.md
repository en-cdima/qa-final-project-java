# qa-final-project-java

![CI Pipeline](https://github.com/en-cdima/qa-final-project-java/actions/workflows/ci.yml/badge.svg)

## Overview
This project is the first stage of a QA Automation Final Project for a Java/Maven-based test automation framework.

It includes:
- Maven project structure
- YAML configuration
- API test logic in pseudocode
- Dockerfile for running Maven tests
- GitHub Actions CI/CD pipeline
- Docker Hub image publishing
